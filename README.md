# shopping_list
Мобильное приложение shopping-list с возможностями создания 10 задач на русском языке (до 25 символов), редактирования и удаления, есть чек-боксы выбора задач. Далее представленна тестовая документация: чек-лист, тест-кейс, баг-репорт.
